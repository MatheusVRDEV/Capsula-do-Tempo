## Projeto
Esse é um projeto  Web responsivo de uma Càpsula do tempo para exibir memorias em uma linha do tempo.

## Tecnologias
Esse Projeto foi desenvolvido durante a NLW da Rocketseat com as seguintes tecnologias:

- HTML
- CSS
- Git e Github
